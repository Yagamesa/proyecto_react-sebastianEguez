# 📝 Proyecto Final - Gestión de Tareas

Aplicación web desarrollada en React para la gestión de tareas (To-Do List), permitiendo crear, editar, eliminar y cambiar el estado de tareas.

## 🚀 Funcionalidades

* ✅ Crear tareas
* ✏️ Editar tareas
* 🗑️ Eliminar tareas
* 🔄 Cambiar estado (Pendiente / Finalizada)
* 🔍 Buscar tareas
* 📂 Filtrar tareas

---

## 🛠️ Tecnologías

* React + TypeScript
* Vite
* Material UI
* Axios
